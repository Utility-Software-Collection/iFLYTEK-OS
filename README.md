# iFLYTEK OS

iFLYTEK OS 是 科大讯飞智慧窗iFLYTEK教育一体机电脑上搭载的系统

系统基于 CMIT_Win 10-v2022-L 优化制作

## 优化内容
1. 修改密码复杂难度 默认账户**admin**无密码
2. 关闭交互式登录
3. 关闭以管理员批准模式运行所有管理员
4. 开启**摄像头**和麦克风权限
5. 恢复**移动热点设置页面**
6. 已完成系统激活

## 预装组件
1. 预装 Windows常用运行组件和运行库
2. 预装.net framework 系统组件
3. REALTEK Bluetooth Driver
4. iFLYTEK Easi Plus
   - iFLYTEK(R) 畅言智慧课堂
   - iFLYTEK(R) 智慧窗管家
   - iFLYTEK(R) 视频展台
